Financial Formula – Net Present Value (NPV)

We evaluate long-term projects using discounted cash flows:

\[ \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} - C_0 \]

  • \(CF_t\): Cash flow at time \(t\)
  • \(r\): Discount rate (cost of capital)
  • \(C_0\): Initial investment
Tie this formula directly to a real project decision the bank might make, e.g., rolling out a new digital lending platform. Highlight that these calculations can be automated and surfaced directly in interactive dashboards.

Data Pipeline – Earnings Summary (Python)


import pandas as pd

# Load quarterly P&L data
df = pd.read_csv("q3_earnings.csv")

# Aggregate key metrics
summary = (
    df.groupby("segment")
      .agg(
          revenue=("revenue", "sum"),
          net_income=("net_income", "sum"),
          roe=("roe", "mean"),
      )
      .reset_index()
)

# Flag segments below ROE threshold
THRESHOLD_ROE = 0.12
summary["underperforming"] = summary["roe"] < THRESHOLD_ROE

print(summary)
